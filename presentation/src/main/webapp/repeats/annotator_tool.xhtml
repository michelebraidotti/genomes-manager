Annotator tool

A java client GUI has been developed to allow AGI annotators and collaborators to easily import and edit repeat sequence data information.

Main features:

  • Bulk import and update using gff3 files.
  • Exports fasta, gff3 and fully annotated gff3 files
  • Single repeat edit
  • Single repeat creation
  • Export flanking regions of identified repeat in fasta format

Currently this tool is in use internally at AGI, we are in the process of releasing it to any collaborator interested. An AGI account is needed in order to use this program.

If you are a collaborator and you want to test this graphical interface please let us know via email to Michele Braidotti or Andrea Zuccolo
